RightArm ペルソナ別ユーザージャーニー

企業×CFOマッチングプラットフォームを訪れたペルソナの反応、感情、行動パターンを詳細に分析

📊 概要

RightArm（企業×CFOマッチングプラットフォーム）を訪れたペルソナの反応、感情、行動パターンを詳細に分析。各ペルソナの心理状態から具体的な行動まで、コンバージョンに至る道筋を明確化しています。

🏢 企業側ペルソナ

35歳 製造業の2代目社長（年商50億）

基本情報： 創業家、経理部ありだが経営管理が弱い、原価把握・部門別管理の強化／資金調達が必要

🎯 サイト訪問時の第一印象

半信半疑
慎重
  • 「また人材紹介サービスか...」（警戒心）
  • 「でも"週1から"って書いてあるな...」（興味）
  • 「完全無料面談まで？本当かな」（疑問）

💭 内心の反応

  • 「CFOってフルタイムで2000万円クラスの人材でしょ？うちにそんな予算はない」
  • 「週1なら月10-20万円くらい？それなら検討できるかも」
  • 「原価計算がグチャグチャなの、誰かに整理してもらいたい」

🔍 サイト内での行動パターン

  1. 検索フィルター使用
    "製造業" "原価管理" "管理会計"でフィルタリング、年収レンジを確認（月80万円〜の表示に「高いな...」）
  2. CFOプロフィール閲覧
    元大手メーカー経理部長のプロフィールを重点的にチェック、"ERP導入実績"の文字に反応
  3. 気になる機能の使用
    3-4人を「気になる」に追加、ただしまだスカウトは送らない（慎重な性格）

😟 離脱リスク要因

  • 「高そう」「敷居が高そう」という先入観
  • 競合他社の事例がないことへの不安
  • 「週1で本当に効果あるの？」という疑問

📈 行動パターン分析サマリー

企業側の共通傾向

  • 初期： 警戒心 → 興味 → 検討 → 行動
  • 重視点： コスト、実績、同業事例、リスクの低さ
  • 決定要因： 具体的成功事例との遭遇

CFO側の共通傾向

  • 初期： 理解 → 評価 → 戦略立案 → 実行
  • 重視点： 専門性の活用、収入最大化、やりがい
  • 決定要因： 自分のスキルにマッチする案件発見

離脱防止策

  • 企業側： 同業事例の充実、料金の透明化、無料相談の強調
  • CFO側： 案件の質向上、マッチング精度向上、手数料の最適化

コンバージョン向上策

  • パーソナライゼーション： 業界・規模別のコンテンツ最適化
  • 社会的証明： 具体的事例・数値の充実
  • リスク軽減： 段階的コミットメントの提案
